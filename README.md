# Emoji Slot

🍇 🍓 🍒 🍐 🍎 🍋 🍍 💎

## Usage

**HTML**

```html
<body>
  ...
  <div id="slot-machine"></div>
  ...
</body>
```

**Javascript**

```js
import EmojiSlot from 'EmojiSlot';
const slotMachine = new EmojiSlot('slot-machine');
```

or

```js
var EmojiSlot = require('EmojiSlot');
var slotMachine = new EmojiSlot('EmojiSlot');
```
